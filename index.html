Open Source Software in the Law Office

By Angel E. Mendoza

What Is Open Source Software?

The Open Source Initiative defines open source software as software that follows the following criteria:
  1. Free redistribution
  2. Easily accessible source code
  3. Allow for derived works
  4. Protect the original author's intent
  5. No discrimination against people

  6. No discrimination against fields of endeavor
  7. Rights apply to all recipients
  8. Not a product-specific license
  9. License can not restrict other software
10. License must be technology-neutral

✦ This definition is used in the State Administrative Manual, Section 4819.2.

What Are the Pros and Cons of Using Open Source Software?

Pros

Open source software offers several advantages over propriety software, such as:
  1. Cost - open source software is typically free to use
  2. Flexibility - users can modify and customize the software to meet their specific needs
  3. Openness - the source code is easily accessible for review
...

Continue...

  4. Security - with many eyes reviewing the code, security flaws are usually found and fixed quickly
  5. Privacy - open source software often collects less data, giving users greater control over their privacy

✦ Open source software’s transparency, security, and privacy help lawyers protect client data and ensure confidentiality, supporting their fiduciary duty.

Cons

While there are many advantages to using open source software, there are also a few disadvantages, such as:
  1. Limited official support, leaving users to rely on community help
  2. Multiple versions or forks which can lead to compatibility issues and confusion
  3. May lack advanced features or polish compared to proprietary software
...

Continue...

  4. Updates and bug fixes often falls on the community or individual users, which can lead to delays or neglect
  5. Troubleshooting and updates can sometimes be difficult and time-consuming

✦ However, most popular open source projects are well-maintained and supported, minimizing these issues.

What Are a Few Examples of Open Source Software?

OnlyOffice (Office Suite)

OnlyOffice allows you to create and edit documents, spreadsheets, presentations, and PDF files. While it may not have all the advanced features of Microsoft Office, it is still powerful enough to edit pleading papers, redact PDFs, and create presentations.

CourtListener (Legal Research Library)

CourtListener is a free legal research website containing millions of legal opinions from federal and state courts.

End Statement

In summary, open source software can be a great option for law offices, offering cost savings, flexibility, and better security. However, it may come with support and compatibility issues. Overall, with the right tools, open source can help lawyers work more efficiently while protecting client data.
